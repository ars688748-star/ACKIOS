# Story 115.13

## Title

Story Lifecycle

## Description

Introduce lifecycle transitions for stories (Planned, In Progress, Completed) and update the workflow automatically during START_STORY and END_STORY.

## Definition of Done

- START_STORY sets Status to In Progress
- END_STORY sets Status to Completed
- START_CHAT displays current Status
- Build PASS
- Tests PASS
