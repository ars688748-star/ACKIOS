# Story 116.28

## Title

Workflow Execution History Persistence

## Description

Improve workflow execution history storage and timeline tracking.

Add persistent history records for every completed workflow story,
including story transitions, commits, execution status and duration.

The dashboard timeline should display real workflow evolution.

## Definition of Done

- Workflow history records stored for every execution
- Story transitions added to history
- Commit information preserved
- Dashboard timeline displays latest executions
- Analytics uses persistent history data
- Build PASS
- Tests PASS
