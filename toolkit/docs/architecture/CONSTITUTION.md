# ACKI OS Constitution

Architecture Rules

BrainKernel
- orchestrates the system
- never implements algorithms

BrainRuntime
- executes runtime pipeline

CognitiveRuntime
- provides cognitive capabilities

Manager
- coordinates domain logic

Engine
- performs algorithms

Builder
- creates object graphs

Bridge
- integrates independent subsystems

Pipeline
- defines execution flow

Stage
- performs one atomic operation

General Rules

- One responsibility per class
- One logical change per commit
- No duplicated managers
- Prefer composition
- Documentation is part of architecture
